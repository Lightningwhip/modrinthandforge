How to Install Mods

1. Download and install Forge from https://files.minecraftforge.net/

2. Download all the mods you want from the links above

3. Press Windows + R, type %appdata%, and press Enter

4. Go to .minecraft folder

5. Create a folder called "mods" if it doesn't exist

6. Move all downloaded mod files (.jar) into the mods folder

7. Launch Minecraft with the Forge profile
